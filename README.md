A custom splash screen for KDE Plasma (originally developed and tested on Nyarch Linux) featuring a smooth animated loop of Kasane Teto.

Inspired by the original `mikuboot` project by [MateRhyu](https://codeberg.org/MateRhyu), adapting the structure to bring everyone's favorite chimera to life.
